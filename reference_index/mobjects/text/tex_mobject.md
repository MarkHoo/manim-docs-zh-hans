# tex_mobject 

Mobjects 表示使用 LaTeX 渲染的文本。

> 重要

> 请参阅相应的教程[Text With LaTeX]()

> 笔记

> 正如您可以使用[`Text`]()（从模块[`text_mobject`]()）向视频添加文本一样，您可以使用[`Tex`]()和[`MathTex`]()插入 LaTeX。
