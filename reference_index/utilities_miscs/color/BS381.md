# BS381

英国颜色标准

该模块包含英国颜色标准之一 BS381C 中定义的颜色。本标准规定了用于识别、编码和其他特殊用途的颜色。请参阅[https://www.britishstandardcolour.com/](https://www.britishstandardcolour.com/) 了解更多信息。

要使用此列表中的颜色，请直接从模块访问它们（该模块暴露于 Manim 的全局名称空间）：

```sh
>>> from manim import BS381
>>> BS381.OXFORD_BLUE
ManimColor('#1F3057')
```
