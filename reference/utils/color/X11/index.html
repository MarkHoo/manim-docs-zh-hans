    X11 颜色

    这些颜色及其名称（取自 https://www.w3schools.com/colors/colors_x11.asp）是马萨诸塞州理工学院 (MIT) 在开发基于颜色的计算机显示系统期间开发的。

    要使用此列表中的颜色，请直接从模块访问它们（该模块暴露于 Manim 的全局名称空间）：

    Bash
    1
2
3
    >>> from manim import X11
>>> X11.BEIGE
ManimColor('#F5F5DC')
