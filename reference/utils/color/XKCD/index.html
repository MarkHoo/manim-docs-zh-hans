    XKCD 颜色名称调查中的颜色

    XKCD 是由 Randall Munroe 创作的流行网络漫画。他的“颜色名称调查”（有 20 万名参与者）得出了近 1000 个颜色名称的列表。

    虽然该XKCD模块暴露于 Manim 的全局名称空间，但其中包含的颜色却不然。这意味着为了使用颜色，请通过模块名称访问它们：

    Bash
    1
2
3
    >>> from manim import XKCD
>>> XKCD.MANGO
ManimColor('#FFA62B')
